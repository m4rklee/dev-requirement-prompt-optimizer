# Release Notes

## v1.0.0

Initial stable packaging release of `dev-requirement-prompt-optimizer`.

### Included capabilities

- guided multi-turn clarification from rough idea to execution-ready prompt
- prompt strengthening for existing development requirement drafts
- hybrid mode for incomplete but already structured prompts
- domain-general section selection instead of a fixed one-size-fits-all outline
- optional UI and interaction specification guidance
- reusable output templates for discovery, synthesis, and final prompt generation
- domain question maps for common software project types

### Packaging additions

- release version file
- installation guide
- one-command install script
- release manifest
- packaged tarball
- checksum file
